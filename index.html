Workstation Activation Steps

Workstation-Aktivierungsschritte

1

Generate Hardware Fingerprint

Hardware-Fingerabdruck generieren

Download and run the software. The Activation screen displays your unique 12-char hardware hash.

Laden Sie die Software herunter und starten Sie diese. Der Aktivierungsbildschirm zeigt Ihren eindeutigen 12-stelligen Hardware-Hash an.

Hardware ID: 1F77A1A2BBE6
2

Complete Purchase

Kauf abschließen

Enter your Hardware ID into the optional checkout field on Lemon Squeezy to automatically pre-register. If you are on mobile, leave it blank; you can bind it later.

Geben Sie Ihre Hardware-ID in das optionale Feld an der Kasse von Lemon Squeezy ein. Wenn Sie mit dem Handy kaufen, lassen Sie das Feld leer; Sie können den Key später zuweisen.

3

Activate and Run Offline

Aktivieren & Offline nutzen

Input your received key into the desktop client. Once verified, the program saves its cache and can operate completely offline on the plant floor.

Geben Sie Ihren Lizenzschlüssel in das Programm ein. Nach erfolgreicher Prüfung arbeitet die Software dauerhaft offline in Ihrer Industrieanlage.
